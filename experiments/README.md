# Historical experiments

The scripts in this directory are preserved as development history and offline research material. They are not supported product entry points, are not invoked by `server.py`, and may rely on a private dataset layout or external services that are not included in this repository.

- `python/`: earlier evaluation, signal-merging, and scoring experiments.
- `swift/`: earlier one-off AI, OCR, and image-collage experiments.

Use the active tools in [`../tools/`](../tools/) and the dashboard workflow instead. Do not assume an experiment is safe to run against a production or shared data directory without reviewing it first.
