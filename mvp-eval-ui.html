필드역할채움추출 방법
입력벡터 = ② 평가 실행에서 함수가 받을 수 있는 신호 — 각 행이 실제로 들고 있는 것. 채움이 낮으면 그 신호를 쓰는 알고리즘의 상한도 낮음 (예: VLM 설명 0% = 아직 못 씀, OCR 44% = 절반만 텍스트 존재).
전체 라이브 버전 = dataset-overview.html (스키마 테이블·툴팁·경고배너·라이브 집계 포함). 이 탭은 그 요약이며 수치는 현재 실제 데이터.

함수 하나가 든 스크립트를 제출하면 eval set 전체에 실행 → 매칭·스코어 → ③ 평가 결과에 새 실행으로 뜬다. 함수는 선택한 입력만 받고 예측 장소명을 반환한다.
